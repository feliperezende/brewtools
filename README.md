# Features
- ABV calculator
- Bottle calculator
- Hydrometer temperature correction
- Unit converter

# Setting up environment
```bash
npm install
```

# How to run

Run on iOS simulator
```bash
npm run ios
```

Run on Android simulator/device
```bash
npm run android
```

Run on web
```bash
npm run web
```
